# blns-java
This is a java implementation for https://github.com/minimaxir/big-list-of-naughty-strings  

## Installation
```xml
<dependency>
  <groupId>com.rasklaad</groupId>
  <artifactId>blns-java</artifactId>
  <version>0.0.1</version>
</dependency>
```

## Usage
```java
NaughtyStrings.getStrings()
```
